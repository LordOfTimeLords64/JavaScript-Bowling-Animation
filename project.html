Principles of Interactive Computer Graphics - Final Project

Bowling Lane by Michael Komnick

Final Project Demo

My project is an animation of a bowling ball hitting pins on a lane. It uses the 3D graphics library THREE.js and the physics library Ammo.js. The user can control the starting position of the ball and the ball's velocity before rolling it. The user can also control the camera with the mouse and arrow keys.

Controls: Press 'a' to move the ball left, press d to move the ball right, press the space bar to roll the ball, and press 'r' to reload the page to its default state. Use the GUI to change the initial velocity of the ball. To see more scene related controls, type '?'.
